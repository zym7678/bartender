# 🍸 调酒库

个人调酒管理工具 — 管理存酒、配方，随机摇一杯，一键制作。

## 功能

- 🍾 **酒柜** — 管理存酒（类型、品牌、开封日期、剩余量）
- 📖 **配方** — 管理鸡尾酒配方（配料、做法、杯型）
- 🎲 **摇一杯** — 随机推荐鸡尾酒，材料不足自动拦截
- 🔍 **做一杯** — 按酒名或基酒搜索配方，一键制作
- 📋 **记录** — 自动记录饮用历史
- ⚡ **PWA** — 可安装到桌面/手机，离线可用

## 在线访问

打开浏览器访问 GitHub Pages 网址即可使用。

## 本地使用

双击 `index.html` 即可在浏览器中打开。
